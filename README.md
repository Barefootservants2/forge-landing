# FORGE v2.1 — Prompt Engineering Application

**Stop Prompting. Start Engineering.**

FORGE transforms ordinary AI prompts into institutional-grade queries using the METATRON v7.2 protocol.

## Overview

FORGE is a prompt engineering tool that coaches users through the CREATE framework to build higher-quality AI prompts before execution. Every output is validated through 11 mandatory quality gates.

## Key Features

### CREATE Framework (Input Scoring)
- **C**ontext — Background, constraints, domain
- **R**easoning — Logic chain, methodology  
- **E**vidence — Source requirements, authority
- **A**ction — Output format, deliverables
- **T**esting — Success criteria
- **E**nhancement — Iteration triggers

### CAKE Standards (Output Validation)
- **C**larity — Unambiguous, structured
- **A**ccuracy — Verified, confidence stated
- **K**indness — Acknowledges uncertainty
- **E**fficiency — Concise, complete

## New in v7.2

- **Claim Registry** — Atomic claim decomposition and tracking
- **Evidence Ledger** — ALCOA+ compliant audit trail with SHA256 hashes
- **Security Gate** — RAG injection protection
- **Independence Score** — Tracks unique primaries vs. echoes
- **RAZIEL Adjudication** — Structured conflict resolution
- **50 Drift Indicators** — Now includes security/integrity monitoring
